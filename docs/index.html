How it works

Two modes. Infinite personality.

Every Proteus cursor starts from one of two base modes. All visual customisation — color, size, shadow, text, presets — is layered on top.

● Circle
shape: 'circle'

Precise. Clean. Iconic.

A perfectly round dot with a smoothly lagging shadow. The shadow follows with a configurable delay — giving depth and elegance. Supports custom size, color, shadow glow, and embedded text.

hover to feel circle mode
◉ Fluid
shape: 'fluid'

Alive. Dynamic. Magnetic.

A blob that squishes and stretches based on mouse velocity. Uses linear interpolation so it always chases the pointer with a satisfying delay. Control physics with speed and maxVelocity.

hover to feel fluid mode

Usage
const cursor = new ProteusCursor({ shape: 'circle' });

// apply preset to live instance (chainable)
cursor.loadPreset('neon');
cursor.loadPreset('chrome', { shape_size: '64px' }); // with overrides

// use preset as constructor base
const cursor2 = new ProteusCursor({
  ...ProteusCursor.getPreset('neon'),
  shape_color: '#ff4444',
});

Quick Start

Up and running in 60 seconds.

Three steps. No build tools required. Works with any framework or vanilla HTML.

Step 1 — Install
# via npm
$ npm install proteuscursor

# or CDN
<link rel="stylesheet"
  href="proteus-cursor.min.css">
Step 2 — Import
// ES Module
import ProteusCursor from
  'proteuscursor';

// or UMD (script tag)
<script src="proteus-cursor.umd.js"></script>
Step 3 — Initialize
// Your first cursor
const cursor = new ProteusCursor({
  shape: 'circle',
  shape_color: '#00D4AA',
  click_animation: 'ripple',
});
