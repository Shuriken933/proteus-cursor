Quick Start

Up and running in 60 seconds

Three steps. No build tools required. Works with any framework or vanilla HTML.

Step 1 — Install
# via npm
$ npm install proteuscursor

# or CDN
<link rel="stylesheet"
  href="proteus-cursor.min.css">
Step 2 — Import
// ES Module
import ProteusCursor from
  'proteuscursor';

// or UMD (script tag)
<script src="proteus-cursor.umd.js"></script>
Step 3 — Initialize
// Your first cursor in 3 lines
const cursor = new ProteusCursor({
  shape: 'circle',
  shape_color: '#00D4AA',
  click_animation: 'ripple',
});

USAGE
const cursor = new ProteusCursor({ shape: 'circle' });

// apply preset to live instance (chainable)
cursor.loadPreset('neon');
cursor.loadPreset('chrome', { shape_size: '64px' }); // with overrides

// use preset as constructor base
const cursor2 = new ProteusCursor({
  ...ProteusCursor.getPreset('neon'),
  shape_color: '#ff4444',
});
Visual Effect

Blend Mode

Apply CSS mix-blend-mode to the cursor shape. The difference value creates automatic color inversion — the cursor is always readable against any background.

dark bg
light bg
new ProteusCursor({
  shape: 'circle',
  shape_size: '40px',
  shape_color: '#ffffff',
  hasShadow: false,
  blend_mode: 'difference',
});

// or at runtime
cursor.setBlendMode('difference');
'normal'

Default. No blending.

'difference'

Inverts colors beneath the cursor. Best with white shape.

'exclusion'

Softer inversion, lower contrast than difference.

any CSS value

Full mix-blend-mode spec is supported.

State Machine

One line of HTML. Infinite states.

Register named cursor states in JS, attach them with a single data-cursor-state attribute.

🖼️
Gallery Item

Expands to 80px with "Zoom" text. Perfect for image grids and portfolios.

▶️
Video Preview
sm-video

Shows ▶ Play at 90px with no shadow. Ideal for video thumbnails.

🎯

sm-cta

Orange tint at 70px with bold "Click". Drives action on key elements.

Integration

Framework Adapters

First-class adapters for React, Vue 3 and Svelte. Each one handles mount/unmount lifecycle automatically and is SSR-safe (Next.js, Nuxt, SvelteKit).

Install and import from the proteuscursor/react subpath:

// Install
npm install proteuscursor

import { useProteusCursor } from 'proteuscursor/react';
import 'proteuscursor/style';

export default function App() {
  const cursor = useProteusCursor({
    shape: 'circle',
    shape_color: '#ffffff',
    click_animation: 'ripple',
  });
  // cursor.current → ProteusCursor instance (null until mounted)
  return <main>...</main>;
}

Use the composable inside <script setup>:

import { useProteusCursor } from 'proteuscursor/vue';
import 'proteuscursor/style';

// inside <script setup>
const cursor = useProteusCursor({
  shape: 'circle',
  shape_color: '#ffffff',
  click_animation: 'ripple',
});
// cursor.value → ProteusCursor instance (null until mounted)

Use inside a <script> block (Svelte 4 & 5):

import { useProteusCursor } from 'proteuscursor/svelte';
import 'proteuscursor/style';

const cursor = useProteusCursor({
  shape: 'circle',
  shape_color: '#ffffff',
  click_animation: 'ripple',
});
// cursor.current → ProteusCursor instance (undefined until mounted)
